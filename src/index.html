Fullscreen Images Plugin

Select an image to make it go fullscreen. You can close it by clicking anywhere on the image or the veil or by using the escape key on your keyboard. I build this to simulate how Google Photos works, when you select an image to enlarge it looks like it grows from the location where the image. This is using Stimulus JS which I saw on best of js and it's simple architecture inspired me to build something.

Check out the source code for this page or you can checkout some of my other work at czaas.dev or on github.
